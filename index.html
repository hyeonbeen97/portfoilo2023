webstandard

이 웹사이트는 HTML5, CSS3, 그리고 JavaScript를 사용하여 제작했습니다.
HTML5는 웹 콘텐츠의 구조를 정의하고, CSS3는 디자인과 레이아웃을 담당하며,
JavaScript는 상호작용과 동적 기능을 구현하는 데 사용하였습니다.
웹사이트 제작 과정에서 웹 접근성도 고려했습니다.
웹 접근성은 모든 사용자가 웹사이트에 쉽게 접근하고
사용할 수 있도록 하는 것을 목표로 하였습니다.

Reactsite

이 사이트는 리액트(React)를 사용하여 제작하였습니다.
리액트는 페이스북에서 개발한 자바스크립트 기반의 프론트엔드 라이브러리로, 사용자 인터페이스를 구축하기 위한 강력한 도구와
컴포넌트 기반 아키텍처를 제공합니다. 제작한 이 사이트는 리액트를 활용하여 모던하고 동적인 사용자 인터페이스를 구현하였습니다.
리액트의 장점을 최대한 활용하여 성능, 유지보수성, 확장성을 고려한 웹사이트를 구축하였습니다. 또한, 리액트를 사용하여 API를 활용한
이 사이트는 동적이고 실시간으로 업데이트되는 컨텐츠를 제공합니다.
